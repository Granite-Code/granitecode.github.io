Welcome to the temporary homepage of Granite.code!

Welcome to the homepage of the Granite.code project. This is an open source project to create a Artificial Intelligence assistant using the Granite AI models from IBM.

So why the Granite models you ask? Well the Granite models are high quality and actively developed models from IBM which also are truly open and free. The Granite models are downloadable and you can run them on your own computer, which means you can get the advantages of a AI Code Assistant, while also protecting your privacy.

The initial version of Granite.code is an extension for the popular VS Code IDE, but over time and with the help of the wider open source community we hope to see it supported in other IDEs too, even Vi and Emacs!

Current features

  • A setup wizard that guides the user through downloading and installing Ollama and the Granite model.
  • Granite models are automatically selected, and updated as new models are released.
  • The UI is tweaked and streamlined for the specific use case.
  • Branding is changed to distinguish our extension.

You can find the full sourcecode for Granite.code here on Github.

The premade Granite.code extension can be downloaded through the VS Code Marketplace
